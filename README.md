## What Is This?

A fully automated n8n workflow that generates **AEO-optimized content** every week вҖ” articles, newsletters, and video scripts вҖ” using a two-model AI pipeline. Built for agencies and content teams who want to rank in both traditional search AND AI answer engines.

**AEO (Answer Engine Optimization)** is the practice of optimizing content so AI systems like ChatGPT, Google AI Overviews, and Perplexity cite your content as a source. This workflow bakes AEO best practices directly into the content generation pipeline.

### Phase 1: Keyword Research (FREE APIs)
- **Google Autocomplete** вҖ” real trending searches (no API key needed)
- **Serper.dev** вҖ” SERP data + People Also Ask questions (2,500 free searches)
- **GPT-4.1** вҖ” analyzes data вҶ’ outputs keyword strategy with AEO angles

### Phase 2: Deep Research
- Fetches **8+ RSS feeds** from industry sources
- **Gemini 2.5 Flash** analyzes all stories, ranks by relevance, extracts data points
- Only includes stories with specific facts (company names, dollar amounts, dates)

### Phase 3: Content Generation (GPT-4.1)
- **AEO Article** (800-1000 words) вҖ” direct-answer opening, question-format headings, FAQ section
- **AEO Editor Pass** вҖ” scores article on 5 criteria, makes targeted optimizations
- **Newsletter Digest** вҖ” 8-12 categorized stories with branded HTML
- **Video Scripts** вҖ” short-form (60s for Reels/TikTok) + long-form (5min for YouTube)

### Phase 4: Email Delivery (4 Branded Emails)
1. **AEO Article** вҖ” full article + `.html` and `.doc` attachments for easy CMS upload
2. **Newsletter Digest** вҖ” categorized stories with inline styling
3. **Video Scripts** вҖ” ready-to-record scripts with visual cues
4. **Instructions** вҖ” step-by-step publishing guide (record videos вҶ’ upload article вҶ’ send newsletter)

### 1. Import the Workflow

1. Open your n8n instance
2. Go to **Workflows** вҶ’ **Import from File**
3. Select `AEO-Content-Engine-v7.0.json`

### 2. Set Up Credentials (3 required + 1 for email)

| Credential | Where to Get | Cost |
|-----------|-------------|------|
| **Serper.dev API Key** | [serper.dev](https://serper.dev) вҶ’ Sign up (no credit card) | FREE (2,500 searches) |
| **OpenAI API Key** | [platform.openai.com](https://platform.openai.com) вҶ’ API Keys | Pay-per-use (~$0.50-1/run) |
| **Google Gemini API Key** | [ai.google.dev](https://ai.google.dev) вҶ’ Get API Key | FREE tier available |
| **SMTP (Gmail)** | Google Account вҶ’ Security вҶ’ App Passwords | FREE |

## Deduplication System

The workflow automatically prevents content repetition:

- **Keyword rotation** вҖ” avoids last 8 used primary keywords
- **Story filtering** вҖ” removes recently featured URLs and titles (4-week window)
- **Angle exclusion** вҖ” GPT is instructed to avoid previously covered angles
- **Title tracking** вҖ” never repeats an article title
- **History retention** вҖ” keeps 52 weeks (1 year) of content history
- **Storage** вҖ” n8n Workflow Static Data (built-in, no external DB needed)

To reset history: run the `RESET History (Manual)` node by itself.
